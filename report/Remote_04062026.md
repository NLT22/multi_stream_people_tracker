## 2. Train FastReID R50 Deploy

Neu chi chay baseline Swin hien tai thi bo qua muc nay.

Prepare crop dataset Market1501-style:

```bash
source venv/bin/activate

python scripts/train_fastreid_mmp.py prepare \
  --install-deps \
  --overwrite \
  --sample-rate 25 \
  --min-w 20 --min-h 40 \
  --min-visible-ratio 0.30
```

Train tren GPU lon:

```bash
python scripts/train_fastreid_mmp.py train \
  --num-gpus 1 \
  --batch-size 128 \
  --test-batch-size 256
```

Neu muon resume:

```bash
python scripts/train_fastreid_mmp.py train \
  --resume \
  --num-gpus 1
```

Eval checkpoint:

```bash
python scripts/train_fastreid_mmp.py eval \
  --weights output/fastreid_mmp/bagtricks_R50_deploy/model_best.pth
```

Export ONNX cho DeepStream/NvMultiObjectTracker:

```bash
python scripts/train_fastreid_mmp.py export \
  --weights output/fastreid_mmp/bagtricks_R50_deploy/model_best.pth \
  --onnx-path models/reid/fastreid_mmp_R50_deploy.onnx \
  --batch-size 16
```

Xoa TensorRT engine cache cu truoc khi test ReID moi:

```bash
python scripts/train_fastreid_mmp.py clean-engines
```

## 5. Nearline Remap Tren Prediction Baseline

`nearline_merge` khong chay detector/tracker. No doc output baseline da export, gom `cam_*_predictions.csv`, `tracklets.csv`, `tracklet_embeddings.npz`, roi sinh ban prediction da remap.

```bash
set -euo pipefail
source venv/bin/activate

SCENES="lobby_0 industry_safety_0 office_0 cafe_shop_0 retail_0"

for SCENE in ${SCENES}; do
  PRED=output/eval/mmp_${SCENE}_nvdcf_realtime_baseline
  OUT=output/eval/mmp_${SCENE}_nvdcf_realtime_baseline_nearline

  python -m src.eval.nearline_merge \
    --pred-dir ${PRED} \
    --out-dir ${OUT} \
    --threshold 0.65 --margin 0.03 \
    --min-gid-embeddings 6 --min-tracklet-detections 10 \
    --mmp-short-root dataset/MMPTracking_short \
    --scene ${SCENE} \
    --geo-weight 0.25 \
    --geo-sample-step 5 \
    --geo-min-overlaps 8 \
    --window-frames 125 \
    --delay-frames 50

  python -m src.eval.metrics_mmp \
    --short-root dataset/MMPTracking_short \
    --scene ${SCENE} \
    --pred-dir ${OUT} | tee output/eval_logs/${SCENE}_nearline.txt
done
```

Expected nearline anchors cu:

| Scene | Baseline Global | Nearline Global | Ghi chu |
|---|---:|---:|---|
| lobby_0 | 0.7649 | 0.8365 | co loi Global split duoc sua |
| industry_safety_0 | 0.8301 | 0.8360 | tang nhe |
| office_0 | 0.8192 | 0.8192 | khong remap |
| cafe_shop_0 | 0.7684 | 0.7684 | khong remap |

## 10. Cach Doc Ket Qua

Uu tien theo thu tu:

1. `Global IDF1`: muc tieu tong quat la `> 0.80`.
2. `Pred IDs`: cang gan GT IDs `7` cang tot, nhung khong duoc wrong-merge.
3. `OVERALL IDF1`: local ID stability theo camera.
4. `MOTA`, `FN`, `FP`: detection/tracking recall/precision.
5. Throughput: pass neu `FPS/cam >= 10` tai so camera muc tieu.

Neu baseline local IDF1 cao nhung Global IDF1 thap, loi chinh la cross-camera association/ReID/geometry. Neu MOTA va recall thap, loi chinh la detection/local tracker/occlusion, nearline kho sua het.
